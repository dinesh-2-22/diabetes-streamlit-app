# Diabetes Prediction App

[Live App Link → Click Here](https://diabetes-app-app-gqd6tc4rrfjymlz3wrxca9.streamlit.app/)

This is a *Diabetes Prediction Web App* built using *Python, **Streamlit, and **scikit-learn*.  
It predicts whether a patient is diabetic based on input features like Glucose, BMI, Age, etc.  

---

## *Features*
- Predicts diabetes using a trained machine learning model.
- Interactive web interface using Streamlit.
- User-friendly input fields for all required features.
- Returns *Prediction*: 0 = Not Diabetic, 1 = Diabetic.

---

## *Installation*
1. Clone the repository:

```bash
git clone https://github.com/dinesh-2-22/diabetes-streamlit-app.git
cd diabetes-streamlit-app

2. Install required packages:



pip install -r requirements.txt 


---

Usage

Run the Streamlit app:

streamlit run app.py

Open the link provided in your browser.

Enter patient details in the input fields.

Click Predict to see the result.
